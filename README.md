# Circuit Supply

Live Website : https://circuitsupply-is.vercel.app

Circuit Supply is a modern eCommerce platform focused on embedded systems and electronics, designed to provide developers, students, and tech enthusiasts with easy access to essential hardware components and tools. The website features a clean, user-friendly interface that allows users to browse products, explore categories, and seamlessly manage their purchases. Built with performance and scalability in mind, it emphasizes fast loading speeds, intuitive navigation, and a smooth shopping experience—key elements for successful online stores that convert visitors into customers. This project demonstrates my ability to develop full-stack, real-world applications that combine practical functionality with a polished user experience.
